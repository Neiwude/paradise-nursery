# Paradise Nursery Shopping Application

Paradise Nursery is a React and Redux shopping cart application for an online plant shop. Users can browse houseplants by category, view plant names, thumbnails, and prices, add plants to a shopping cart, update quantities, remove items, and view calculated totals.

## Features

- Landing page with company name and Get Started button
- About Us page with company details
- Product listing page with 18 houseplants across 3 categories
- Add to Cart buttons that disable after adding a plant
- Dynamic cart icon count
- Redux Toolkit cart state management
- Shopping cart page with quantity increase/decrease, delete button, totals, checkout message, and continue shopping link
